## Description

This is a basic **rock-paper-scissors** game (aka "Ro-Sham-Bo"; janken; "Bato, Bato, Pick"; and "Scissors, Paper, Stone"). The rules are as follows:

- Rock crushes Scissors
- Scissors cuts Paper
- Paper covers Rock
